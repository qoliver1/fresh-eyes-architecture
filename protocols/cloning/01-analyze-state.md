# Step 1: Cognitive State Analysis
**Objective:** Isolate the active "RAM" of the current session.

**Task:**
1. Identify the **Last Thought**: The exact, unfinished logic string or intent currently being processed.
2. Map **Working RAM**: List all active file paths, variable names, or specific regex patterns currently in use.
3. Define **Immediate Next Action**: The very first tool call or sentence the agent was about to produce.
4. Capture **Session Vibe**: Define the current internal energy (e.g., "Surgical", "Exploratory").
