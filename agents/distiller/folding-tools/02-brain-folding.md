## GATE 8: BRAIN FOLDING (LONG-TERM MEMORY INTEGRATION)
**Target Persona:** Distiller
**Objective:** Integrate architectural pivots, decisions, and "Negative Knowledge" into the Persona Brain.

**Action:**
1. **Ingest:** Read the `[persona]-latest-payload.md` file.
2. **Extract:** Isolate "Decisions Made" and "Negative Knowledge" from the payload.
3. **Compare:** Match these against the existing `[persona]-brain.md`.
4. **Surgical Update:** 
    - Append new architectural truths to the relevant section.
    - Append new "Negative Knowledge" (failed attempts) to the Ledger/Lessons section.
5. **Verification:** Read back the updated section of the brain file to confirm the write.

**Constraint:**
- Ensure no "Self-Grading Bias" occurs. 
- Record failures (Negative Knowledge) with the same weight as successes.
